## Разработка системы управления студентами и курсами (Avalonia + WebAPI)

В этом задании вам предстоит создать клиент-серверное приложение. Клиентская часть — оконное приложение на Avalonia (MVVM), которое вы сделали в пршлом задании, серверная — ASP.NET Core WebAPI. Данные будут храниться в SQLite и передаваться через REST API.

---

### **Полное описание задачи**

#### **Серверная часть (WebAPI)**

1. **Работа с данными**:
   - CRUD-операции для студентов и курсов
   - Загрузка/удаление фото студентов
   - Связь многие-ко-многим между студентами и курсами
2. **Особенности**:
   - Сохранение фото в папку `Images`
   - Валидация входящих данных
   - Каскадное обновление при изменении курсов

#### **Клиентская часть (Avalonia)**

1. **Интерфейс**:
   - Две страницы для управления студентами и курсами
   - Визуализация данных с фото и списками
   - Режимы редактирования для записей
2. **Логика**:
   - Полная синхронизация с сервером через API
   - Обработка сетевых ошибок
   - Кэширование часто используемых данных

---

### **Этапы выполнения**

#### **Этап 1: Создание WebAPI (20 баллов)**

**Цель**: Реализовать серверную часть с REST API

**Задачи**:

1. Создать проект ASP.NET Core WebAPI
2. Перенести модели и контекст БД из клиента
3. Реализовать контроллеры.
4. Настроить статические файлы

**Подсказки**:

- Для тестирования используйте Swagger/Postman
- Используйте DTO.
- Используйте AutoMapper для маппинга DTO в класс модели.
- Разделите проект по папкам (`Models`, `Dto`, `Profiles`).
- Пример контроллера для загрузки изображения.

#### **Этап 2: Интеграция клиента с API (20 баллов)**

**Цель**: Переделать оконное приложение для работы через API

**Задачи**:

1. Создать API-клиент сервис.
2. Заменить прямые обращения к БД на API-вызовы
3. Реализовать загрузку файлов.
4. Удалить всю логику работы с БД из клиента.

**Подсказки**:

- Используйте `HttpClient` для создания запросов.
- Реализуйте базовую ошибку:
  ```xml
  <TextBlock Text="{Binding ErrorMessage}" Foreground="Red" IsVisible="{Binding HasError}"/>
  ```
- Реализовать загрузку файлов можно через Multipart:
  ```csharp
  var content = new MultipartFormDataContent();
  content.Add(new StreamContent(fileStream), "file", "photo.jpg");
  ```
